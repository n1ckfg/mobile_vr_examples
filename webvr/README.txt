Three helpful tools:
1. Clear Cache Chrome Extension
https://chrome.google.com/webstore/detail/clear-cache/cppjkneekbjaeellbfkmgnhonkkjfpdn?hl=en
Clear cache and refresh page with one click.

2. WebVR API Emulation Chrome Extension
https://chrome.google.com/webstore/detail/webvr-api-emulation/gbdnpaebafagioggnhkacnaaahpiefil?hl=en
Emulate Vive hardware. (Do not use with normal desktop/mobile polyfill.)

3. Node.js
https://nodejs.org/en/
To preview locally using the included shell scripts, install this, then type from the command line:
npm install -g httpserver
From now on you can see a local preview of your work by just double-clicking the included shell script for your OS and setting your browser to:
localhost:8080
